Projects

Optical Character Recognition

During my time in New Zealand and with the tech job opportunities during Covid, I worked for the New Zealand Electoral Commission doing data entry. This was a tedious manual process of inputting the data from handwritten forms into a database. I built a tool using the Google Vision API to read the handwritten forms and inject the data into the database. This improved form processing time by ~10x.

Backr Maps

The project I was working on in Roam was a product called Backr. We wanted to display some key metrics to the team in a visually appealling way to motivate us and show our impact. I used the Google Maps API to pull sign-up data from our analytics platform and plotted it on a map of Australia, showing where each new user was signing up from every day.
